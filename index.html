2023夏算是又有一段空余时间了，就把丢低了，但又一直都挺想做的个人沙雕网友 blog 重新捡起来。

本来嘛，还想再深入折腾一下，自己长期租一个服务器来 host 这个个人 blog 的。但鉴于我日常三分钟热度，大概率不会长期在这里写东西了，那就还是直接部署在 github 上吧，毕竟谁能拒绝白嫖呢。

那么在这里你能看到的东西，除了一部份正经的！！我在工作上的，和自己瞎折腾的技术积累以外，还会夹杂着一些沙雕网友发言和一堆 R18 的奇怪 XP 宣言，所以请尽量在没有人的地方阅读。

虽然大概率也没人看就是了。。。
